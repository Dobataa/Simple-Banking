# Simple Banking System

A small banking system built with TypeScript and Node.js.

## Features

- Create accounts:
  - Checking Account (with overdraft limit)
  - Savings Account (with interest rate)

- Operations:
  - Deposit money
  - Withdraw money (with daily limit)
  - Transfer between accounts
  - Apply interest (Savings accounts only)

- Validation:
  - No negative deposits
  - Overdraft rules enforced
  - Daily withdrawal limits
  - Maximum balance for savings accounts
